## 一、应用整体风格

### 1.1 整体风格定义 Prompt

```
一款温暖有趣的宠物内心解读应用。面向年轻养宠群体（18-35岁）。整体风格活泼可爱但不幼稚，配色以暖橙色(#FF8C42)为主色调，搭配奶油白(#FFF9F0)背景。使用圆润的无衬线字体，所有按钮采用全圆角胶囊形状。界面留白充足，氛围轻松愉悦。
```

### 1.2 主题配色 Prompt

```
将主色调设为暖橙色(#FF8C42)，用于主要按钮和品牌元素。辅助色为暖黄色(#FFD166)，用于强调和图标。背景使用奶油白(#FFF9F0)，卡片为纯白色。文字主色深灰(#333333)，次要文字中灰(#888888)。
```

### 1.3 字体与按钮风格 Prompt

```
使用圆润友好的无衬线字体。所有按钮采用全圆角胶囊形状(pill shape)。卡片组件使用16px圆角和轻微阴影。输入框使用浅灰边框和8px圆角。
```
